Для установки и использования в программе БАЗИС-Мебельщик необходимо:
1. Скопировать папку с текстурами в директорию: C:\Users\"Пользователь"\Documents\Bazis\Текстуры
2. Импортировать материалы EGGER: Настройка -> База материалов; Выберите Файл-> Импорт номенклатуры из XML. В открывшемся окне в поле "Имя файла" укажите путь к файлу Egger.xml, в строке "Проверять уникальность материала" выберите чекбокс "По артикулу и наименованию", нажмите "OK".
3. Назначить в базе материалов группам материалов тип: Правая кнопка мыши по группе (в структуре), в выпавшем меню выбрать "Установить группе тип материала":
   а) "Древоплитная продукция" и "Столешницы" устанавливаем тип - Листовой и погонный материалы.
   б) "Кромка" - Кромочный материал.
